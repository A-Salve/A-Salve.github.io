# A-Salve.github.io

Учебный многостраничный сайт **Smart Deals Daily** для практики деплоя на сервер.

## Страницы
- `/index.html` — главная
- `/catalog.html` — каталог категорий
- `/blog.html` — блог с публикациями
- `/contact.html` — контакты
- `/posts/2025-08-10.html` и `/posts/2025-08-17.html` — страницы постов

## Локальный запуск
```bash
python3 -m http.server 8000
```
